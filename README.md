# 📊 Análise de Dados de Vendas com Python

## 📌 Visão Geral do Projeto
Este projeto realiza uma análise exploratória de dados (EDA) em um conjunto de dados de vendas no varejo, com o objetivo de gerar insights de negócio, identificar padrões e oportunidades de melhoria.

## 🎯 Objetivos
- Identificar os produtos mais vendidos
- Analisar a receita por região
- Detectar tendências de vendas ao longo do tempo
- Encontrar oportunidades de crescimento no negócio

## 🛠️ Ferramentas e Tecnologias
- Python
- Pandas
- Matplotlib
- Seaborn

## 📈 Principais Insights de Negócio

- A região West concentra a maior parte da receita total, indicando forte presença de mercado
- Existe sazonalidade clara com pico de vendas nos meses 11 e 12 (período de fim de ano)
- Alta dependência de poucos produtos líderes, representando risco de concentração de receita
- A região South apresenta baixo desempenho, sugerindo oportunidade de expansão comercial
